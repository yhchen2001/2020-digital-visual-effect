Feature detection

使用Haris corner detector尋找每一張照片中的local maximum R 首先對原始照片處裡成黑白的影像以及resize照片來加速處裡，處裡完的圖片先經過一層gaussian filter來減少照片雜訊，接著藉由上課說明的Talor expansion方法，找出$Ix^2$ $Iy^2$和$Ixy$得到matrix M，再用M的det值和trace值找出照片中每一點的R值 為了避免找到的feature 在同樣的範圍之內，用一個maximum filter對R找出一個kernel範圍內的最大值，並且只保留這個最大的R，同時也刪除一些照片邊界的feature(減少誤差還有方便descriptor處裡)

Feature description

找MaxR附近的點形成一個factor(此次作業找出$7*7$的範圍，做成一個49長度的vector)來descript每一個local maximum feature，並且將每一個vector normalize(避免亮度差造成影響)

Feature matching

將兩兩圖片的所有descriptor做inner product，找到一組最大的pair，且值要過threshold以及比第二大的pair高出一定值

Warping

跟投影片上說的一樣，我們是先弄了一張新的，跟原圖shape一樣的全黑圖片，再把對應過來的xy設定好。最後在把圖片左右的黑邊切掉。 focal length是我們用試出來效果最好的那個

Ransac

找好matching point之後就可以開始做ransac。 我們的做法是如果matching pairs超過RANSAC_K的話，就隨機取RANSAC_K個點來測試。如果沒有超過的話就每個點都試試看。

假設左圖叫A，右圖叫B，先找到一組matching point (Ay0, Ax0), (By0, Bx0)，然後找到 (dy0, dx0) = (Ay0 - By0, Ax0 - Bx0)。再把所有的(Byi, Bxi) 都加上 (dy, dx)，看有多少個點是inlier（跟相對的 (Ayi, Axi) 的距離在RANSAC_THRES_DIS以內）

這個會找到P-1個(dy, dx)

Blending and Stitching

這個部分我們做了Naive, linear blending on edge, linear blending on overlap。

我們寫了一個blend的function，傳入的參數是兩張圖片 left_img, right_img，回傳的圖片是同樣大小，全部都被blend好的。

colC = colA * (w-1 - i)/w + colB * (i+1)/w
